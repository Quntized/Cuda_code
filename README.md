# Cuda_code
this repo contains all the example cases of code for defining section. You can find the section wise code (which are not contains in absolute form) . Here you will find the full running example code while understanding the behind the scene.

This repo also contains a section modeled on `kokkos/core/src/Cuda/`, written
in my own way for two purposes: first, to explore whether any real internal
issues exist in that area; second, as a way for me to learn the underlying
patterns Kokkos uses in its CUDA backend. That's what I enjoy most - dissecting
complex systems to understand how they actually work. Thanks.

[CUDA Programming Guide](https://docs.nvidia.com/cuda/cuda-programming-guide/index.html) - for reading. 
[Kokkos cuda folder](https://github.com/kokkos/kokkos/tree/develop/core/src/Cuda) - Kokkos's work contain in CUDA section.
